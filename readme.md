# Workspace - Typescript

```
$ npm install -D typescript
```

Build to js

```
$ ./node-modules/.bin/tsc <filename.ts>
# or
$ npx tsc <filename.ts>

$ node <filename.js>
```

Run Typescript via ts-node

```
$ npm install -D ts-node
```

```
$ ./node-modules/.bin/ts-node <filename.ts>
# or
$ npx ts-node <filename.ts>
```
